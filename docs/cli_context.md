# beaker context

The `beaker context` commands manage contexts within a Beaker project and inspect the contexts installed in the current environment.

| Command | Purpose |
|---|---|
| `beaker context new` | Scaffold a new context inside the current Beaker project. |
| `beaker context list` | List the contexts installed in the current environment. |

> This page is a placeholder. Run `beaker context --help` for the authoritative list of options.
