# beaker notebook

`beaker notebook` starts a local Beaker notebook server. Once it is running, the UI is available in your browser at [`localhost:8888`](http://localhost:8888).

```bash
beaker notebook
```

Related: `beaker dev` starts a development server with file watching and auto-reload, useful when iterating on a custom context.

> This page is a placeholder. Run `beaker notebook --help` and `beaker dev --help` for the authoritative list of options.
