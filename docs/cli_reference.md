# CLI Reference

The `beaker` command-line tool covers everything from launching a notebook to scaffolding new context and subkernel projects. The pages in this section document each command group. For an authoritative, always-current view, run any command with `--help`:

```bash
beaker --help
beaker context --help
beaker subkernel --help
```
