# Getting Started

This section covers everything you need to get a working Beaker environment up and running, from installing the base package to bringing in the contexts and subkernels you want to use.

If this is your first time with Beaker, follow these pages in order:

1. **[Installing Beaker](install.html)** — installs the base `beaker-kernel` package and the `beaker` CLI.
2. **[Adding Contexts](adding_contexts.html)** — installs prebuilt contexts published as Python packages (for example, a domain-specific context shared by your team).
3. **[Adding Subkernels](adding_subkernels.html)** — installs prebuilt subkernels that add language support beyond the built-in Python, R, and Julia.
4. **[Your First Notebook](your_first_notebook.html)** — a quick tour of the notebook interface and the agent.

If you want to *build* a context, subkernel, or integration rather than install one, head over to [Development](development.html).
