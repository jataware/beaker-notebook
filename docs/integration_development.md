# Integration Development

Integrations let a context expose external resources (APIs, datasets, libraries, skill bundles) to the agent. This section covers writing your own integration providers and the REST API the frontend uses to interact with them.

If you have not yet read the conceptual overview, see [Key Concepts > Integrations](concept_integrations.html) first.
