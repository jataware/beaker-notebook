# Key Concepts

These pages introduce the vocabulary you will encounter while using Beaker — in the UI, in conversation with the agent, and in documentation written by others. Each page is a short conceptual introduction, not a how-to: it answers "what is this thing and what is it doing for me." For mechanics and how to build your own, see [Development](development.html).
